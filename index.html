Two modes

Pick your level
of friction.

Different people need different walls. Switch any time from the popup.

STRICT
Hard block
Sites are blocked at the network level. The page never loads. Chrome shows a generic error. There's no "are you sure" button to talk yourself past. Best for anyone serious about staying off gambling sites.
SOFT
Warning overlay
A warning appears before content loads. You can go back, or override and continue if you're sure. Useful for accountability with flexibility, or for trying things out before committing to strict.

FAQ

Common questions.

Is it really free?
Yes. There's no premium tier, no trial, no paywall. The extension does everything it does for free, forever. If you want to support the project, there's a coffee button. Everything else is free.
What if I find a gambling site that isn't blocked?
Open Settings in the popup and add it to your custom blocklist. It'll be blocked immediately. You can also report it on GitHub so it gets added to the main blocklist for everyone.
Can I disable it?
Yes — Chrome lets you disable any extension at chrome://extensions. This isn't enterprise lockdown software. The point is to add friction, not to make blocking impossible. If you want a wall you can't climb, ask a friend to install it on a profile only they have the password for.
Does it work in Incognito?
By default Chrome disables extensions in Incognito. To enable: go to chrome://extensions, find Anti-Gambling Shield, click "Details," and toggle "Allow in Incognito."
Why was this built?
The browser is where the temptation lives. One push notification, one autocomplete suggestion, one ad, and the urge is back. Removing the surface area where gambling sites can reach you is one of the highest-leverage things you can do. This extension does that, and nothing else.
